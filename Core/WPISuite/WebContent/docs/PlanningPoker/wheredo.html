Where do I start?

When the planning poker module is first opened, you are viewing the Overview Panel. In order to get back to the overview panel, click on the Current Sessions tab, next to the options tab. From the overview panel, you can access the navigation buttons, at the top of the screen, and you can access the tree of sessions, below and to the left. To the right of the tree of sessions is the panel that will show you detailed information about the session you have selected within the tree. From the overview panel, you can create a session, edit a session, vote on a session, view session statistics, end voting on a session, make final estimates on a session, and archive a session. In order to learn more about these options, please click on the corresponding node in the left tree panel.
